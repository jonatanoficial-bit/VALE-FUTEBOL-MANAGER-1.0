# Selection Mode 2026 — Base de dados desta build

## Base usada nesta etapa
- Ranking FIFA masculino oficial com atualização de 01/04/2026
- Lista Top 50 reproduzida em formato legível pela ESPN em 01/04/2026
- Confirmação de seleções classificadas para a Copa do Mundo 2026 no site da FIFA

## Decisão de design
Nesta build, o jogo converte ranking real em **overall jogável** e em **travas de Career Score**.
Isso foi feito para o modo seleção ficar divertido e escalável antes da camada completa de convocações reais.

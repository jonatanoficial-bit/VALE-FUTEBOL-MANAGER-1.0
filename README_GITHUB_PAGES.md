# Vale Futebol Manager 2026 (build v1.12.8)

## GitHub Pages (sem dor de cabeça)
Este pacote já vem com uma cópia completa do site dentro da pasta **/docs**.

1) No GitHub: Settings → Pages
2) Source: **Deploy from a branch**
3) Branch: **main** (ou master) / Folder: **/docs**
4) Salvar

Assim os assets (fundo, logos, CSS/JS) não dão 404.

Se você preferir publicar pela raiz (/), também funciona (os arquivos estão na raiz e em /docs).
